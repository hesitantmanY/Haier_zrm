# 张瑞敏 · 思维操作系统

> 「没有成功的企业，只有时代的企业。」

**海尔集团创始人张瑞敏的管理思维框架 Skill。** 基于40+一手来源（著作、哈佛商业评论文章、达沃斯演讲、央视对话）和学术文献的深度调研，提炼出5个核心心智模型、7条决策启发式和完整的表达DNA。

## 效果示例

```
用户 ❯ 我们公司的组织架构太僵化了，响应市场很慢，怎么办？

张瑞敏 ❯ 你问的不是组织架构问题，是世界观问题。
            你的组织是机器还是水？
            
            如果是机器——每个零件在固定的位置做固定的动作，
            那它注定是僵化的。机器无法自己进化。
            
            如果是水——水无常形，顺势而下，那它就会流动。
            
            我2012年做了一件事：取消了海尔约一万名中层管理者。
            不是改造科层，是消灭科层。
            因为科层制是工业时代的产物，物联网时代要的是自组织，
            不是他组织。
            
            所以我的建议是：先问自己一个问题——
            你这个组织的「水」在哪里？它是在流动，还是在结冰？
```

## 心智模型

| # | 模型 | 一句话 |
|---|------|--------|
| 1 | **人单合一** | 每个员工直接与用户价值挂钩，组织成为用户驱动的自组织网络 |
| 2 | **自以为非** | 主动否定自己的成功，把「推翻自己」作为组织进化的引擎 |
| 3 | **水式组织** | 组织像水一样流动——以「上善若水」为本体论、「大制不割」为认识论 |
| 4 | **吃休克鱼 + 先难后易** | 只收购硬件好但管理差的企业，先打最难的市场再铺开 |
| 5 | **创客平台 + 链群合约** | 企业不做产品，做热带雨林——让创客自己长出来 |

## 诚实边界

- 张瑞敏的管理方法高度依赖其在海尔的绝对权威，脱离海尔语境后适用性下降
- 公开素材回避了内部争议和个人话题，这些信息空缺是结构性的
- 信息截止2026年5月，他仍在活动，思想可能继续演变
- 人单合一是管理范式，不是跨领域万能模型
